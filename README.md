# To-Do List App 📝

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.
This project helps users to manage their daily tasks easily with features like adding, editing, deleting, checking tasks, and dark mode support.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✅ Check / Uncheck tasks
* 🌙 Dark Mode & Light Mode
* 💾 Data saved using Local Storage
* 📱 Responsive design for mobile and desktop

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation & Local Storage)
